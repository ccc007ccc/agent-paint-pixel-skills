# Modern Clean Pixel Icon

Use this style for app icons, badges, tool symbols, UI glyphs, simple inventory items, and documentation icons.

## Visual DNA

- Simple geometry.
- Small palette.
- Minimal texture.
- Reads clearly at 1x.

## Palette

- Use 4-10 colors.
- Use one base, one shadow, one highlight, and one accent if needed.
- Avoid near-identical shades.

## Layer Order

Top to bottom:

```text
symbol_detail
single_highlight
single_shadow
base_shape
outline_or_silhouette
```

## APX Guidance

- Use full rows for 16x16 icons.
- Use chunks for small symbols or marks.
- Patch by simplifying colors and removing extra detail.

## Do

- Favor iconic shape.
- Keep corners and curves intentional.
- Use high contrast between symbol and background.

## Avoid

- Dense internal texture.
- Dithering.
- Details that need zoom to read.
