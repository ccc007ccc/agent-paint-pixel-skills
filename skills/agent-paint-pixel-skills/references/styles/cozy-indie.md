# Cozy Indie

Use this style for farming, food, animals, gentle UI, cute props, soft creatures, and relaxed scenes.

## Visual DNA

- Warm palette.
- Rounded forms.
- Soft colored outlines.
- Gentle contrast and friendly details.

## Palette

- Use 8-16 colors.
- Prefer warm low-contrast ramps.
- Use dark brown, purple, or green outlines instead of pure black.

## Layer Order

Top to bottom:

```text
sparkle_or_effects
cute_details
soft_highlight
warm_shadow
base
colored_outline
soft_shadow
```

## APX Guidance

- Use full rows for simple small animals and foods.
- Use chunks for cheeks, eyes, shine, and tiny accessories.
- Patch harshness by changing black outlines to colored outlines.

## Do

- Keep shapes rounded and readable.
- Use small highlights and friendly face details.
- Use restrained texture.

## Avoid

- Harsh black outlines everywhere.
- Cold saturated shadows.
- Gritty noise.
